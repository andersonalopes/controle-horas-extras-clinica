# Controle de Horas Extras – Clínica Médica

Projeto de Análise de Dados e Business Intelligence voltado ao controle de horas extras em uma clínica médica.

## Objetivo
Apoiar a gestão no controle de horas extras, custos operacionais e identificação de setores com maior impacto financeiro.

## Tecnologias
- Power BI
- SQL / MySQL
- CSV
- GitHub

## Análises Realizadas
- Total de horas extras
- Custo financeiro por cargo e setor
- Funcionários com maior volume de horas extras
- Motivos mais recorrentes de horas extras  

## Resultado
Dashboard interativo para apoio à tomada de decisão estratégica.
